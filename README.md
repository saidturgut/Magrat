# magrat

**magrat** is a machine monitor that can simulate all kinds of computers. it currently uses cli as view

#

**x8bit :** this machine hosts the 8 bit microchip cpus from the 70s

- mos 6502
- intel 8080
- zilog z80
- motorola 6809 (in progress)

**pdp11 :** (planned)
